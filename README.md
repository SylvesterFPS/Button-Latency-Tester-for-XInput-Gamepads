# Button-Latency-Tester-for-XInput-Gamepads
I built my own gamepad latency tester. My goal is to measure the actual Controller to PC latency you experience while playing games

**Requirements:**
- Teensy 4.0 / 4.1 Microcontroller
- Micro USB Cable
- Teensyloader from PJRC: https://www.pjrc.com/teensy/loader_win10.html (to install the .hex file for the Teensy)
- Firmware (.hex file) for Teensy (Download the .zip file)
- XInput Latency Tester Software (Download the .zip file)

**How to wire:**
- Teensy 4.0 / 4.1 over USB to PC
- your XInput Gamepad over USB to PC
- connect Pin 0 from the Teensy to the A-Button on your Controller/Gamepad (You need to disassembling the controller)
**- Now start the Test**
